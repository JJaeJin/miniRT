# [Build and Execute]
## Build
```bash
git clone git@github.com:JJaeJin/miniRT.git
cd miniRT
make
```
Then, you can see `miniRT` file.
## Execute
To execute miniRT, you need formatted file with `.rt` expansion.
```bash
./miniRT example.rt
```
## Test Environment
- OS: macOS Catalina 10.15.7
- g++: Apple clang version 12.0.0
- library: minilibX
